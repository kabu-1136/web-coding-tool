🎯 HTML（骨組み）の穴埋めミッション

抜けているタグや文字をエディタで探して、正しく補ってみましょう。

  • タイトル： <head> の中に、<title>東海大学付属相模高等学校</title> を追加しましょう。
    ※ 反映されると、右側プレビューの「ブラウザのタブ」が変化します！
  • 学校名を大見出しに： <header> の中に、<h1>東海大学付属相模高等学校</h1> を追加しましょう。
  • ナビにリンクを追加： <nav> の中に、<a href="#about">学校概要</a> を追加しましょう。
  • 表の完成： 2つ目の <tr> 内に、所在地（<th>所在地</th><td>神奈川県相模原市南区相南3-33-1</td>）を入れましょう。
  • 部活名を追加： 運動部と文化部の下にある空の <p></p> に部活動の名前（例：野球部、吹奏楽部など）を書きましょう。

🎨 CSS（見た目）の穴埋めミッション

右上の「CSSを適用する」トグルをオンにして、style.css タブを切り替えて以下のプロパティを追加してみましょう。

  • 背景色（body）を追加： 全体の背景を薄いグレーにするため、body 内に background-color: #f8f9fa; を追加しましょう。
  • ヘッダーをおしゃれに： header 内に、背景色 background-color: #00479e; （東海ブルー）と、テキスト中央寄せ text-align: center; を追加しましょう。
  • ヘッダー文字サイズ： header h1 内に font-size: 2rem; を、header p 内に opacity: 0.8; を追加しましょう。
  • 見出しのデザイン調整： h2 内に font-size: 1.5rem; を追加し、さらに h2::after の中の空いている width: に 50px; を追記しましょう。
  • 小見出しの左線： h3 内に、左側の線を作るための border-left: 4px solid #00479e; を追加しましょう。
  • フッターの文字色： footer の中に文字色を白にする color: #ffffff; を追加しましょう。
